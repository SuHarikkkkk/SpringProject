Cosmetics Store  

Веб-приложение интернет-магазина косметики, разработанное с использованием Java Spring Boot.  
Проект реализует базовую функциональность онлайн-магазина: просмотр каталога товаров, просмотр цен, добавление товаров в заказ и оформление покупки.  
Пользователь может просматривать каталог товаров, получать информацию о продукте, выбирать товары и оформлять заказ.  

Основные возможности:  
• Просмотр каталога косметических товаров  
• Просмотр информации о товаре  
• Отображение цен на товары  
• Добавление товаров в заказ  
• Оформление заказа  
• Просмотр списка заказов  

Технологии:  
• Java 25  
• Spring Boot  
• Spring Web  
• Spring Data JPA  
• Hibernate  
• REST API  
• SQL Database  

Структура проекта (основные сущности):  
• Product — товар (название, описание, цена, категория)  
• Order — заказ пользователя  
• OrderItem — товары внутри заказа  
• Category — категория косметики
